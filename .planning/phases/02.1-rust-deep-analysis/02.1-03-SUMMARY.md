## Decisions Made
- coverage_requested flag (single-frame trigger) instead of spawning thread directly in toolbar keeps toolbar stateless; draw_panels.rs owns scan_msg_tx and handles the actual spawn
- BFS traverses import edges in both directions so both importers and importees of the selected file are highlighted
- Risk normalization computes max_raw once per frame at the top of the render pass, propagates via RenderContext — keeps color functions pure with no global state
