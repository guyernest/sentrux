<task type="auto" tdd="true">
  <name>Task 2: Implement snapshot writer, loader, delta computation, and pruning</name>
  <files>sentrux-core/src/analysis/snapshot_writer.rs, sentrux-core/src/analysis/mod.rs</files>
  <behavior>
    - write_analysis_snapshot creates .sentrux/snapshots/{epoch}.json with valid AnalysisSnapshot JSON
    - load_nearest_snapshot returns None when snapshots dir is empty
    - load_nearest_snapshot returns the closest snapshot that predates the target epoch
    - load_nearest_snapshot ignores snapshots newer than target
    - prune_snapshots keeps at most 50 files, deleting oldest first
    - compute_delta_report produces correct FileDeltaEntry for files that improved, regressed, or stayed same
    - compute_delta_report handles files present in baseline but not current (and vice versa)
  </behavior>
  <action>
    Create sentrux-core/src/analysis/snapshot_writer.rs with these functions:

    1. `write_analysis_snapshot(root: &str, pmat: &Option<PmatReport>, coverage: &Option<CoverageReport>, clippy: &Option<ClippyReport>) -> Result<String, String>`:
       - Build AnalysisSnapshot from current reports (iterate pmat files, look up coverage/clippy per path)
       - computed_at = epoch_now()
       - Create dir: std::fs::create_dir_all(root/.sentrux/snapshots)
       - Write JSON to {root}/.sentrux/snapshots/{epoch}.json
       - Call prune_snapshots(root, 50) after write
       - Return Ok(path) or Err(message)

    2. `load_nearest_snapshot(root: &str, target_epoch: i64) -> Option<AnalysisSnapshot>`:
       - Read .sentrux/snapshots/ dir entries
       - Parse filenames as {epoch}.json
       - Find the largest epoch <= target_epoch
       - Deserialize and return that snapshot
       - Return None if no qualifying snapshot exists

    3. `compute_delta_report(baseline: &AnalysisSnapshot, pmat: &Option<PmatReport>, coverage: &Option<CoverageReport>, clippy: &Option<ClippyReport>) -> TimelineDeltaReport`:
       - Build a HashMap from baseline files (path -> FileAnalysisSnapshot)
       - For each file in current reports, compute deltas using grade_delta() for TDG, subtraction for coverage_pct, subtraction for clippy_count
       - Files only in current (not baseline) get no delta entry (new file, no comparison)
       - Files only in baseline (not current, deleted) get no delta entry
       - Return TimelineDeltaReport { by_file, baseline_epoch: baseline.computed_at }

    4. `prune_snapshots(root: &str, max_count: usize)`:
       - List .sentrux/snapshots/*.json
       - If count > max_count, sort by epoch ascending, delete oldest until count == max_count

    Add `pub mod snapshot_writer;` to sentrux-core/src/analysis/mod.rs.

    - test_load_nearest_empty_dir: returns None
    - test_load_nearest_all_newer: all snapshots newer than target returns None
    - test_prune_snapshots: write 5 snapshots, prune to 3, verify 2 oldest deleted
    - test_compute_delta_improved: file goes from C to A, delta is positive
    - test_compute_delta_regressed: file goes from A to C, delta is negative
