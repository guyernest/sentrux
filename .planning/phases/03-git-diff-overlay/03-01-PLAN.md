  <action>
    1. In `sentrux-core/src/metrics/evo/git_walker.rs`, add:
       - `DiffWindow` enum with `TimeSecs(i64)`, `CommitCount(u32)`, `SinceLastTag` variants
       - Derive `Debug, Clone, Copy, PartialEq, serde::Serialize, serde::Deserialize`
       - `impl Default for DiffWindow { fn default() -> Self { DiffWindow::TimeSecs(86400) } }`
       - `DiffWindow::PRESETS` const array with 7 entries: (TimeSecs(900), "15m"), (TimeSecs(3600), "1h"), (TimeSecs(86400), "1d"), (TimeSecs(604800), "1w"), (SinceLastTag, "tag"), (CommitCount(1), "1c"), (CommitCount(5), "5c")
       - `DiffWalkResult` struct with `records: Vec<CommitRecord>`, `new_file_paths: HashSet<String>`
       - `pub(crate) fn walk_git_log_windowed(root: &Path, window: DiffWindow) -> Result<DiffWalkResult, String>`:
         - Reuse existing Repository::discover, revwalk setup, scan_root_prefix
         - For TimeSecs: use epoch cutoff (same logic as walk_git_log)
         - For CommitCount: track count, break after N commits processed
         - For SinceLastTag: call find_last_tag_epoch, use its result as epoch cutoff (fall back to epoch 0 if no tags = walk entire history)
         - Track new files via `delta.status() == git2::Delta::Added` (NOT old_file check)
         - Reuse existing parse_commit/collect_diff_files, skip merge commits and mega-commits
       - `fn find_last_tag_epoch(repo: &Repository) -> Result<i64, String>`:
         - Use `repo.tag_names(None)` to list all tags
         - For each tag, `repo.find_reference("refs/tags/{name}")` then `reference.peel_to_commit()` (handles both lightweight and annotated tags)
         - Return the highest epoch among all tags, or Err if no tags exist
       - Make CommitRecord and CommitFile `pub` (not just pub(crate)) since git_diff_adapter needs them from another module. Or re-export via mod.rs.

       - `fn compute_git_diff_report(root: &str, window: DiffWindow) -> Result<GitDiffReport, String>` — calls walk_git_log_windowed, builds GitDiffReport::from_walk
       - `pub fn save_analysis_snapshot(root: &str, snapshot: &AnalysisSnapshot) -> Result<(), String>` — writes `.sentrux/snapshot.json`
       - `pub fn load_snapshot_at_boundary(root: &str, boundary_epoch: i64) -> Option<AnalysisSnapshot>` — walks git history via git2 to find snapshot blob at/before boundary, limit to 1000 commits checked
